Saya lahir dan besar di tanah Papua karena kedua orang tua saya menikah dan bekerja disana. Saya 2 bersaudara dan saya sebagai anak pertama. Setelah selesai sekolah Alhamdulillah saya lolos SNMPTN 2019 dan diterima di Universitas Negeri Malang dengan program studi S1 Teknik Informatika dan saat ini sudah mencapai semester 7.
